Indentinator!
=============

Indentinator is a Ruby script that inspects and changes the indentation in
your source files.

Usage 
=====
  
    indentinator [-c INDENT_AMOUNT] <file> [<file2>...]
  
Options
-------
  
    -c    Modify the indentation amount to INDENT_AMOUNT
    -v    Verbose output
    -V    Very verbose output
